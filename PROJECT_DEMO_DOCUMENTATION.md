
**演示步骤：**

```
后台检测更新 → 新版本就绪 → 显示更新提示 → 一键更新
```

## 🚀 快速开始

### 本地运行

```bash
# 克隆项目
git clone https://github.com/your-org/hailan.git

# 安装依赖
cd hailan
npm install

# 启动开发服务器
npm run dev

# 访问
http://localhost:5173
```

### 生产构建

```bash
# 构建
npm run build

# 预览
npm run preview

# 部署
# 将 dist 目录部署到服务器
```

### PWA测试

```bash
# 1. 构建生产版本
npm run build

# 2. 启动预览服务器
npm run preview

# 3. 在Chrome中打开
# F12 → Application → Service Workers
```
