## 1. 设计系统和核心框架

### 1.1 设计语言系统

#### 1.1.1 颜色系统

```
主要颜色:
- 海蓝蓝 (#0056b3) - 品牌主色,营造平静、高端、科技感
- 深夜灰 (#1a1a1a) - 背景色,提供舒适的阅读体验
- 极光紫 (#6B46C1) - 强调色,用于交互元素
- 医疗白 (#FFFFFF) - 背景色,强调健康和清洁

次要颜色:
- 活力珊瑚 (#ED8936) - 强调按钮,增强视觉吸引力
- 清新绿 (#38A169) - 成功状态,健康导向
- 柔和粉 (#FBB6CE) - 温馨提醒,情感连接
- 警告黄 (#ECC94B) - 注意提醒
- 错误红 (#F56565) - 错误提醒

语义颜色:
- 成功绿 (#48BB78) - 操作成功
- 警告黄 (#ECC94B) - 注意事项
- 错误红 (#F56565) - 错误状态
- 信息蓝 (#0056b3) - 信息提醒

中性色:
- 从纯白到纯黑的12级灰度
- 确保WCAG 2.1 AA标准
- 支持深色模式切换
```

#### 1.1.2 字体系统

```
字体家族:
- 英文: Inter 字体家族 (Regular, Medium, SemiBold)
- 中文: 苹方-简 (系统默认字体)
- 数字: SF Pro Display (用于价格、数据)

字号层级:
- 12px - 辅助文字、标签
- 14px - 正文、描述文字
- 16px - 小标题、按钮文字
- 18px - 标题、重要信息
- 20px - 大标题
- 24px - 页面主标题
- 30px - 模态框标题
- 36px - 超大标题
- 48px - 品牌展示

行高:
- 1.2 - 紧凑行高(小字号)
- 1.4 - 标准行高(正文)
- 1.6 - 宽松行高(标题)
- 1.8 - 超宽松行高(大标题)
```

#### 1.1.3 间距系统

```
基础单位: 8px

间距层级:
- 4px - 元素内部间距
- 8px - 小间距、相关元素间距
- 12px - 标准间距、卡片内边距
- 16px - 中等间距、块间距
- 24px - 大间距、主块间距
- 32px - 超大间距、页面块间距
- 48px - 超大间距、页面边距
- 64px - 页面顶部/底部间距
- 96px - 区块间距
```

#### 1.1.4 圆角系统

```
圆角层级:
- 小圆角: 4px - 按钮、输入框、标签
- 中圆角: 8px - 卡片、模态框
- 大圆角: 16px - 大卡片、容器
- 全圆角: 50% - 头像、图标、圆形按钮
```

#### 1.1.5 阴影系统

```
级别1: 0 1px 3px rgba(0,0,0,0.12) - 微妙阴影
级别2: 0 4px 6px rgba(0,0,0,0.1) - 标准阴影
级别3: 0 10px 15px rgba(0,0,0,0.1) - 卡片阴影
级别4: 0 20px 25px rgba(0,0,0,0.1) - 浮动阴影
```

## 2. 核心页面流程设计

### 2.1 引导和注册流程

#### 2.1.1 欢迎页面 (1440x900)

```
设计元素:
- 简洁的品牌Logo展示
- 标语:"智能亲密生活,隐私优先体验"
- 两个主要入口:
  * "开始探索" - 进入商城
  * "隐私说明" - 了解隐私保护
- 底部年龄确认声明
- 动画: 优雅的淡入效果
```

#### 2.1.2 隐私模式选择页面 (移动端 375x812)

```
设计元素:
- 三个隐私级别选项:
  a) 访客模式(眼睛图标,临时浏览)
     - 描述: 无需注册,浏览基础产品
  b) 轻量账户(云图标,基础功能)
     - 描述: 仅邮箱,无个人身份信息
  c) 隐私账户(锁图标,全功能加密)
     - 描述: 端到端加密通信,本地数据存储选项
- 每个选项的简要描述
- 视觉隐私级别指示器
- 默认选择: 隐私账户
```

#### 2.1.3 注册流程(多步骤)

```
步骤1: 邮箱/手机输入
- 带隐私提示
- 验证码发送
- 进度指示器: 1/5

步骤2: 密码设置
- 显示强度指示器
- 密码要求描述
- 进度指示器: 2/5

步骤3: 偏好调查(可选)
- 可视化问卷
- 场景选择
- 健康需求
- 进度指示器: 3/5

步骤4: 隐私设置确认
- 可调节设置
- 默认隐私选项
- 进度指示器: 4/5

步骤5: 欢迎界面
- 个性化问候
- 快速开始指南
- 进度指示器: 5/5
```

#### 2.1.4 权限请求模态框

```
设计元素:
- 位置权限(可选,用于配送)
  - 清晰的"为什么需要"描述
  - 地图预览
- 通知权限(可选,智能提醒)
  - 推送类型描述
- 相机权限(可选,AR功能)
  - 本地图片处理描述
- 每个权限都有清晰描述
- 允许/拒绝按钮
```
