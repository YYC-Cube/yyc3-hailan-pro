## 一、组件库概述

### 1.1 组件库结构

```
HaiLan-Components-CN/
├── 基础组件 (Atomic Components)
│   ├── 按钮 (Buttons)
│   ├── 输入框 (Inputs)
│   ├── 图标 (Icons)
│   └── 文字 (Typography)
├── 复合组件 (Molecular Components)
│   ├── 卡片 (Cards)
│   ├── 导航 (Navigation)
│   ├── 表单 (Forms)
│   └── 反馈 (Feedback)
├── 组织组件 (Organism Components)
│   ├── 页面头部 (Page Headers)
│   ├── 页面底部 (Page Footers)
│   ├── 模态框 (Modals)
│   └── 列表 (Lists)
└── 布局组件 (Layout Components)
    ├── 容器 (Containers)
    ├── 网格 (Grids)
    └── 间隔 (Spacers)
```

### 1.2 命名规范

**中文组件命名格式**：

```
组件类型-功能描述-状态

示例：
Button-Primary-提交-Default
Input-Text-搜索框-Focus
Card-Product-产品卡片-Empty
Modal-Confirm-确认对话框-Open
```

## 七、组件状态管理

### 7.1 组件状态定义

**默认状态 (Default)**：组件的初始状态
**悬停状态 (Hover)**：鼠标悬停时的状态
**激活状态 (Active)**：组件被激活时的状态
**焦点状态 (Focus)**：组件获得焦点时的状态
**禁用状态 (Disabled)**：组件被禁用时的状态
**加载状态 (Loading)**：组件正在加载时的状态
**错误状态 (Error)**：组件出现错误时的状态
**空状态 (Empty)**：组件无内容时的状态

### 7.2 状态转换规则

**按钮状态转换**：

```
Default → Hover → Active → Default
Default → Disabled → Default
Default → Loading → Default
```

**输入框状态转换**：

```
Default → Focus → Default
Default → Error → Focus → Default
Default → Disabled → Default
```

**卡片状态转换**：

```
Default → Hover → Default
Default → Empty → Default
```
