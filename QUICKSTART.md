# HaiLan Pro - 快速开始

## 🚀 最快部署方式

### 方式1: v0一键发布（30秒）

1. 点击v0界面右上角 **"Publish"** 按钮
2. 等待部署完成
3. 访问生产URL

**就这么简单！**

---

## 💻 本地开发

### 前置要求

- Node.js 18+
- npm 或 pnpm

### 3步启动

```bash
# 1. 克隆项目
git clone https://github.com/YYC-Cube/yyc3-hailan-pro.git
cd yyc3-hailan-pro

# 2. 安装依赖
npm install --legacy-peer-deps

# 3. 启动开发服务器
npm run dev
```

访问: http://localhost:3000

---

## 📦 可用命令

```bash
npm run dev          # 启动开发服务器（端口3000）
npm run build        # 构建生产版本
npm run preview      # 预览生产构建（端口4173）
npm run test         # 运行测试
npm run typecheck    # TypeScript类型检查
npm run deploy       # 构建并部署到Vercel
```
