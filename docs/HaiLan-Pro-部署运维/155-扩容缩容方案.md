### 3. 扩容缩容方案

#### 3.1 自动扩容策略

**HPA 配置：**

```yaml
apiVersion: autoscaling/v2
kind: HorizontalPodAutoscaler
metadata:
  name: hailan-frontend-hpa
spec:
  scaleTargetRef:
    apiVersion: apps/v1
    kind: Deployment
    name: hailan-frontend
  minReplicas: 2
  maxReplicas: 10
  metrics:
  - type: Resource
    resource:
      name: cpu
      target:
        type: Utilization
        averageUtilization: 70
  - type: Resource
    resource:
      name: memory
      target:
        type: Utilization
        averageUtilization: 80
  behavior:
    scaleDown:
      stabilizationWindowSeconds: 300
      policies:
      - type: Percent
        value: 50
        periodSeconds: 60
    scaleUp:
      stabilizationWindowSeconds: 0
      policies:
      - type: Percent
        value: 100
        periodSeconds: 30
```

#### 3.2 手动扩容

**扩容命令：**

```bash
# 扩容到 5 个副本
kubectl scale deployment hailan-frontend --replicas=5 -n hailan-pro

# 查看 ReplicaSet 状态
kubectl get rs -n hailan-pro
```

#### 3.3 集群扩容

**节点自动扩容 (Cluster Autoscaler)：**

```yaml
apiVersion: cluster.k8s.io/v1beta1
kind: MachineDeployment
metadata:
  name: hailan-worker
spec:
  replicas: 3
  minReplicas: 2
  maxReplicas: 10
  template:
    spec:
      providerSpec:
        instanceType: c5.xlarge
```
