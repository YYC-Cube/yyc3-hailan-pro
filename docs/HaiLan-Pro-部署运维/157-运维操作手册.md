## 核心内容

### 1. 背景与目标

#### 1.1 项目背景
HaiLan Pro (海蓝) 是新一代高端、私密、智能的情趣健康生活管理平台。项目基于「五高五标五化」理念，通过 PWA 技术结合 AI 智能辅助与物联网，为用户提供从生理健康到心理愉悦的全方位解决方案。

#### 1.2 项目愿景
打造极致隐私、智能陪伴、品质合规、全场景覆盖的情趣健康生活管理平台，为用户提供安全、专业、高端的健康生活体验。

#### 1.3 核心价值主张
- **极致隐私**：双重加密、隐私浏览模式及伪装发货机制
- **智能陪伴**：基于 LLM 的 AI 情感与生理健康顾问
- **品质合规**：医疗级标准商品，高端"海蓝蓝"视觉调性
- **全场景覆盖**：PWA 端支持离线浏览、桌面安装及无缝推送

#### 1.4 文档目标

### 3. 运维操作手册

#### 3.1 日常运维检查

**每日检查清单：**

- [ ] 检查集群健康状态
- [ ] 查看告警信息
- [ ] 检查备份任务执行
- [ ] 查看资源使用趋势
- [ ] 检查证书有效期

**检查命令：**

```bash
# 集群健康检查
kubectl cluster-info
kubectl get componentstatus

# 节点状态检查
kubectl get nodes -o wide

# Pod 状态检查
kubectl get pods -A | grep -v Running

# 资源使用检查
kubectl top nodes
kubectl top pods -A
```

#### 3.2 版本升级流程

**滚动升级步骤：**

```bash
# 1. 更新镜像版本
kubectl set image deployment/hailan-frontend \
  frontend=registry.hailan.pro/hailan-frontend:v1.1.0 \
  -n hailan-pro

# 2. 查看升级状态
kubectl rollout status deployment/hailan-frontend -n hailan-pro

# 3. 如需回滚
kubectl rollout undo deployment/hailan-frontend -n hailan-pro

# 4. 查看历史版本
kubectl rollout history deployment/hailan-frontend -n hailan-pro
```

#### 3.3 应急响应流程

**P0 级故障响应：**

1. **0-5分钟**：确认故障、通知团队
2. **5-15分钟**：初步排查、确定范围
3. **15-30分钟**：执行修复、验证效果
4. **30-60分钟**：根因分析、制定预防
5. **事后复盘**：更新文档、优化流程
