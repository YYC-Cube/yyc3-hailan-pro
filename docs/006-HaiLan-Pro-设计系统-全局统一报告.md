## 八、设计原则 ✅ 已统一

### 8.1 核心原则

1. **隐私至上**：所有设计优先考虑用户隐私
2. **简洁优雅**：界面简洁，避免过度设计
3. **一致性**：全局保持视觉和交互一致性
4. **可访问性**：确保所有用户都能使用
5. **性能优先**：动画和效果不影响性能

### 8.2 设计规范

1. **禁用黑色**：全局不使用纯黑（#000000）
2. **大圆角风格**：统一使用24px大圆角
3. **毛玻璃效果**：核心组件使用Backdrop Blur
4. **流畅动效**：所有过渡使用标准时长和缓动
5. **响应式设计**：完美适配所有设备
