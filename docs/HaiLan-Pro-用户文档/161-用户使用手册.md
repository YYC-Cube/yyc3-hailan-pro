# 161 用户使用手册

## 核心内容

### 1. 背景与目标

#### 1.1 项目背景
HaiLan Pro (海蓝) 是新一代高端、私密、智能的情趣健康生活管理平台。项目基于「五高五标五化」理念，通过 PWA 技术结合 AI 智能辅助与物联网，为用户提供从生理健康到心理愉悦的全方位解决方案。

#### 1.2 项目愿景
打造极致隐私、智能陪伴、品质合规、全场景覆盖的情趣健康生活管理平台，为用户提供安全、专业、高端的健康生活体验。

#### 1.3 核心价值主张
- **极致隐私**：双重加密、隐私浏览模式及伪装发货机制
- **智能陪伴**：基于 LLM 的 AI 情感与生理健康顾问
- **品质合规**：医疗级标准商品，高端"海蓝蓝"视觉调性
- **全场景覆盖**：PWA 端支持离线浏览、桌面安装及无缝推送

#### 1.4 文档目标
- 规范用户使用手册相关的业务标准与技术落地要求
- 为项目相关人员提供清晰的参考依据

### 3. 用户使用手册

#### 3.1 快速入门

欢迎使用海蓝(HaiLan Pro)！海蓝是一款高端、私密、智能的情趣健康生活管理平台。本手册将帮助您快速上手并充分利用平台的各项功能。

**第一步：注册账户**

1. 访问海蓝官网：`https://hailan.pro`
2. 点击页面右上角的「注册」按钮
3. 填写注册信息：
   - 邮箱地址：用于登录和接收重要通知
   - 设置密码：建议包含字母、数字和特殊字符，长度不少于8位
   - 确认密码：再次输入密码确保一致
4. 选择您的初始隐私等级：
   - **标准模式**：正常显示所有信息，适合私密环境使用
   - **隐身模式**：敏感信息自动脱敏，适合共享设备使用
   - **伪装模式**：虚拟化显示所有信息，适合最高隐私需求
5. 阅读并同意《用户协议》和《隐私政策》
6. 点击「注册」完成注册

**第二步：验证邮箱**

注册成功后，系统将向您的邮箱发送一封验证邮件。请在24小时内点击邮件中的验证链接完成邮箱验证。验证后即可享受完整的平台功能。

**第三步：完善个人信息**

登录后，建议您完善个人信息以获得更个性化的服务：

1. 点击右上角头像，选择「个人中心」
2. 设置昵称：您希望在平台显示的名称
3. 上传头像：支持JPG、PNG格式，建议尺寸200x200像素
4. 设置收货地址：用于购买商品的配送
5. 完善健康档案：帮助我们提供更精准的健康建议

#### 3.2 隐私模式使用指南

海蓝提供三种隐私模式，您可以根据使用场景随时切换：

##### 标准模式 (STANDARD)

适合在私密环境中使用，所有信息正常显示。

**特点：**
- 完整显示用户名、邮箱、手机号
- 显示完整的订单历史和浏览记录
- 正常显示商品名称和描述

**切换方法：**
进入「设置」→「隐私设置」→ 选择「标准模式」

##### 隐身模式 (STEALTH)

适合在公共场合或共享设备上使用。

**特点：**
- 邮箱显示为 `***@hailan.pro`
- 手机号显示为 `138****8000`
- 浏览历史自动隐藏
- 订单商品名称虚拟化显示

**切换方法：**
进入「设置」→「隐私设置」→ 选择「隐身模式」→ 确认切换

##### 伪装模式 (DISGUISE)

提供最高级别的隐私保护。

**特点：**
- 用户昵称虚拟化显示
- 所有订单内容完全虚拟化
- 收货地址使用虚拟地址
- APP图标和名称伪装

**切换方法：**
进入「设置」→「隐私设置」→ 选择「伪装模式」→ 输入二次确认密码

#### 3.3 智能商城使用指南

##### 浏览商品

海蓝商城采用场景导向分类体系，帮助您快速找到合适的产品：

**场景分类：**
- **CARE呵护**：日常健康护理产品
- **PLAY乐趣**：情趣娱乐产品
- **INTIMACY亲密**：亲密关系产品
- **LINGERIE内衣**：性感内衣服饰
- **SMART智能**：智能硬件设备

**浏览步骤：**
1. 点击底部导航栏「商城」
2. 选择感兴趣的场景分类
3. 滑动浏览商品列表
4. 点击商品图片查看详情

##### 商品筛选与搜索

**筛选功能：**
- 按价格区间筛选
- 按品牌筛选
- 按功能特性筛选
- 按评分筛选

**搜索功能：**
1. 点击顶部搜索框
2. 输入关键词（商品名称、功能、品牌等）
3. 查看搜索结果
4. 使用筛选条件进一步缩小范围

##### 商品详情

点击商品卡片进入详情页，您可以：

- 查看商品大图和详细介绍
- 阅读用户真实评价
- 查看规格参数
- 选择颜色、尺寸等规格
- 添加到购物车或立即购买

##### 购物车管理

**添加商品：**
1. 在商品详情页选择规格
2. 点击「加入购物车」
3. 可继续购物或进入购物车结算

**管理购物车：**
1. 点击底部导航栏「购物车」
2. 修改商品数量
3. 删除不需要的商品
4. 选择收货地址
5. 点击「去结算」

#### 3.4 订单管理

##### 下单流程

1. **确认订单信息**
   - 收货地址：可选择已有地址或添加新地址
   - 商品清单：核对商品、规格、数量
   - 运费信息：查看运费标准

2. **选择支付方式**
   - 微信支付
   - 支付宝
   - 银行卡支付

3. **提交订单**
   - 确认无误后点击「提交订单」
   - 完成支付

##### 订单状态查询

订单状态包括：
- **待付款**：订单已创建，等待支付
- **待发货**：支付完成，商家准备发货
- **待收货**：商品已发出，等待签收
- **已完成**：订单完成
- **已取消**：订单已取消
- **退款中**：退款处理中

查看订单：
1. 点击底部导航栏「我的」
2. 点击「我的订单」
3. 查看订单列表和详情

##### 伪装发货

海蓝提供伪装发货服务，保护您的隐私：

- 快递单不显示任何敏感信息
- 快递公司使用普通寄件人信息
- 快递外包装为普通纸箱
- 发送短信时内容虚拟化

#### 3.5 AI健康顾问使用指南

海蓝AI健康顾问是基于大语言模型开发的智能助手，为您提供专业的健康咨询和建议。

##### 开启AI对话

1. 点击底部导航栏「AI助手」
2. 输入您的问题或需求
3. 点击发送按钮

##### 对话主题

AI助手可以帮您解决以下类型的问题：

**健康咨询：**
- 生理健康问题
- 心理情感问题
- 生活建议

**产品推荐：**
- 根据您的需求推荐合适产品
- 解答产品使用问题
- 对比不同产品特点

**使用指导：**
- 产品使用方法
- 注意事项
- 常见问题解答

##### 对话技巧

- **描述清晰**：尽量详细描述您的问题
- **循序渐进**：可以分多次提问，逐步深入了解
- **尊重隐私**：AI对话内容完全保密

#### 3.6 PWA应用安装

海蓝支持PWA(渐进式Web应用)技术，可以像原生应用一样安装到您的设备上。

##### iOS安装

1. 使用Safari浏览器访问 `https://hailan.pro`
2. 点击底部「分享」按钮
3. 向下滚动，点击「添加到主屏幕」
4. 点击「添加」完成安装

##### Android安装

1. 使用Chrome浏览器访问 `https://hailan.pro`
2. 点击浏览器菜单（三个点）
3. 点击「添加到主屏幕」或「安装应用」
4. 确认安装

##### PWA功能优势

- **离线访问**：无网络时也可浏览已缓存内容
- **推送通知**：及时接收订单状态、活动优惠等通知
- **全屏体验**：沉浸式使用体验
- **自动更新**：自动获取最新版本
