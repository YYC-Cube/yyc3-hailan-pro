### 3. 常见问题FAQ

#### 3.1 账户相关问题

##### Q1: 如何注册海蓝账户？

**A:** 您可以以下方式注册：
1. 邮箱注册：输入邮箱地址、设置密码
2. 手机注册：输入手机号、获取验证码
3. 第三方登录：支持微信、支付宝快捷登录

注册后请验证邮箱/手机号，以确保账户安全。

##### Q2: 忘记密码怎么办？

**A:** 密码找回步骤：
1. 在登录页点击「忘记密码」
2. 输入注册邮箱或手机号
3. 获取验证码
4. 设置新密码

**注意事项：**
- 新密码不能与原密码相同
- 建议使用字母+数字+符号的组合

##### Q3: 如何修改个人信息？

**A:** 修改入口：
1. 进入「我的」→「个人中心」
2. 点击「编辑资料」
3. 修改相应信息
4. 点击「保存」

**可修改内容：**
- 昵称
- 头像
- 性别
- 生日
- 简介

**不可修改内容：**
- 注册邮箱/手机号（安全考虑）
- 用户ID

##### Q4: 如何注销账户？

**A:** 注销账户步骤：
1. 进入「我的」→「设置」
2. 点击「账户安全」
3. 滚动到底部点击「注销账户」
4. 阅读注销须知
5. 输入密码确认注销

**注销前请注意：**
- 未完成订单将自动取消
- 会员权益无法保留
- 数据永久删除无法恢复
- 建议先导出重要数据

#### 3.2 订单相关问题

##### Q5: 如何查看订单状态？

**A:** 查看订单步骤：
1. 点击底部导航「我的」
2. 点击「我的订单」
3. 查看订单列表和详情

**订单状态说明：**
- 待付款：订单创建，等待支付（30分钟内）
- 待发货：已支付，商家准备发货
- 待收货：已发货，等待签收
- 已完成：订单完成
- 已取消：订单已取消
- 退款中：退款处理中

##### Q6: 如何取消订单？

**A:** 订单取消规则：
- 待付款状态：可直接取消
- 待发货状态：联系客服取消
- 已发货状态：无法取消，可申请退货

**取消步骤：**
1. 进入订单详情页
2. 点击「取消订单」
3. 选择取消原因
4. 确认取消

##### Q7: 如何申请退款？

**A:** 退款申请步骤：
1. 进入订单详情页
2. 点击「申请退款」
3. 选择退款原因
4. 上传凭证（如需要）
5. 提交申请

**退款时效：**
- 审核时间：1-3个工作日
- 退款到账：3-7个工作日

**退款条件：**
- 未发货订单：全额退款
- 已发货订单：需要退货后退款
- 定制商品：不支持退款

##### Q8: 发票怎么开？

**A:** 发票开具步骤：
1. 进入订单详情页
2. 点击「开具发票」
3. 填写发票信息：
   - 发票类型：电子发票/纸质发票
   - 发票抬头：个人/公司
   - 税号：公司抬头需要填写
4. 确认提交

**注意事项：**
- 电子发票：订单完成后7天内可申请
- 纸质发票：随货寄出或单独邮寄
- 发票开具后不支持更换

#### 3.3 支付相关问题

##### Q9: 支持哪些支付方式？

**A:** 支持的支付方式：
- 微信支付
- 支付宝
- 银行卡支付（借记卡/信用卡）
- 余额支付

##### Q10: 支付失败怎么办？

**A:** 常见原因和解决方法：

| 失败原因 | 解决方法 |
|----------|----------|
| 余额不足 | 充值或更换支付方式 |
| 支付密码错误 | 重置支付密码 |
| 超过限额 | 分批支付或更换支付方式 |
| 网络问题 | 检查网络后重试 |
| 银行限制 | 联系银行解除限制 |

##### Q11: 支付成功但订单状态未更新？

**A:** 可能是网络延迟导致：
1. 等待1-2分钟刷新页面
2. 检查支付账单是否扣款
3. 如已扣款订单未更新，联系客服处理
4. 提供订单号和支付凭证

#### 3.4 隐私相关问题

##### Q12: 隐私模式有什么区别？

**A:** 三种隐私模式区别：

| 模式 | 邮箱显示 | 手机号显示 | 浏览记录 | 商品显示 |
|------|----------|------------|----------|----------|
| 标准模式 | 完整显示 | 完整显示 | 完整记录 | 真实名称 |
| 隐身模式 | 脱敏显示 | 脱敏显示 | 不记录 | 虚拟化 |
| 伪装模式 | 虚拟化 | 虚拟化 | 不记录 | 完全虚拟 |

##### Q13: 伪装发货真的安全吗？

**A:** 是的，伪装发货是我们的核心隐私保护服务：
- 快递单无敏感信息
- 快递外包装普通纸箱
- 发货人信息虚拟化
- 收到货前无法辨认内容

##### Q14: 切换隐私模式会影响数据吗？

**A:** 不会影响数据：
- 切换模式仅改变数据显示方式
- 原始数据保持不变
- 切换回原模式数据恢复正常
- 可以随时自由切换

#### 3.5 物流配送问题

##### Q15: 发货时间和配送时效？

**A:** 配送时效说明：

| 商品类型 | 发货时间 | 配送时效 |
|----------|----------|----------|
| 现货 | 24小时内 | 1-3天 |
| 预售 | 7天内 | 1-3天 |
| 定制 | 15天内 | 1-3天 |

**偏远地区：**
- 新疆、西藏、青海等地区配送时效延长3-5天

##### Q16: 如何查询物流信息？

**A:** 物流查询步骤：
1. 进入订单详情页
2. 点击「查看物流」
3. 查看最新物流信息
4. 点击快递单号跳转到快递官网

##### Q17: 可以修改收货地址吗？

**A:** 地址修改规则：
- 未发货：可以修改
- 已发货：无法修改，联系客服尝试拦截
- 自提点订单：不支持修改

#### 3.6 退换货问题

##### Q18: 退换货政策是什么？

**A:** 海蓝退换货政策：

**7天无理由退货：**
- 商品保持原样
- 包装完整
- 不影响二次销售
- 非定制商品

**不支持退货：**
- 内裤等贴身物品
- 定制商品
- 已使用商品
- 特价商品

##### Q19: 如何申请退货？

**A:** 退货申请步骤：
1. 进入订单详情页
2. 点击「申请退货」
3. 选择退货原因
4. 上传商品照片
5. 提交申请
6. 等待审核（1-3天）
7. 审核通过后寄回商品
8. 仓库收货后退款

##### Q20: 退货运费谁承担？

**A:** 运费承担规则：
- 商品质量问题：商家承担
- 个人原因退货：买家承担
- 会员权益：金卡及以上会员免费上门取件

- 客服电话：400-XXX-XXXX

**A:**

**A:** 解决方法：
1. 检查系统通知权限是否开启
2. 检查应用通知设置
3. 检查勿扰模式
4. 重新登录账户

##### Q24: 如何退出所有设备登录？

**A:** 安全设置步骤：
1. 进入「我的」→「设置」
2. 点击「账户安全」
3. 点击「退出所有设备」
4. 确认操作
