# 163 AI伴侣使用指南

## 核心内容

### 1. 背景与目标

#### 1.1 项目背景
HaiLan Pro (海蓝) 是新一代高端、私密、智能的情趣健康生活管理平台。项目基于「五高五标五化」理念，通过 PWA 技术结合 AI 智能辅助与物联网，为用户提供从生理健康到心理愉悦的全方位解决方案。

#### 1.2 项目愿景
打造极致隐私、智能陪伴、品质合规、全场景覆盖的情趣健康生活管理平台，为用户提供安全、专业、高端的健康生活体验。

#### 1.3 核心价值主张
- **极致隐私**：双重加密、隐私浏览模式及伪装发货机制
- **智能陪伴**：基于 LLM 的 AI 情感与生理健康顾问
- **品质合规**：医疗级标准商品，高端"海蓝蓝"视觉调性
- **全场景覆盖**：PWA 端支持离线浏览、桌面安装及无缝推送

#### 1.4 文档目标
- 规范AI伴侣使用指南相关的业务标准与技术落地要求

### 3. AI伴侣使用指南

#### 3.1 AI伴侣简介

海蓝AI伴侣是基于先进大语言模型(LLM)技术开发的智能健康顾问，为您提供7x24小时的专业咨询和陪伴服务。

**核心功能：**

| 功能 | 说明 |
|------|------|
| 健康咨询 | 生理健康、心理健康专业咨询 |
| 产品推荐 | 智能推荐适合您的健康产品 |
| 使用指导 | 产品使用方法、注意事项指导 |
| 情感陪伴 | 倾听您的烦恼，提供情感支持 |
| 知识科普 | 健康知识、科学普及 |

#### 3.2 开启AI对话

##### 首次使用

1. 点击底部导航栏「AI助手」
2. 阅读AI使用须知
3. 点击「开始对话」
4. 输入您的第一个问题

##### 对话界面介绍

```
┌─────────────────────────────────┐
│    AI伴侣          [设置] [清空] │
├─────────────────────────────────┤
│                                 │
│  对话区域                       │
│  - 显示历史对话                 │
│  - AI回复和您的消息             │
│                                 │
├─────────────────────────────────┤
│  [输入您的问题...]      [发送]  │
└─────────────────────────────────┘
```

##### 语音输入

1. 点击输入框右侧的麦克风图标
2. 说出您的问题
3. 系统自动转换为文字并发送

**支持语言：**
- 中文（普通话）
- 英语
- 粤语（部分功能）

#### 3.3 对话主题指南

##### 健康咨询类

**可以咨询的问题：**
- "我最近睡眠质量不太好，有什么建议吗？"
- "如何改善亚健康状态？"
- "什么运动适合久坐办公族？"
- "如何缓解工作压力？"

**AI会：**
- 了解您的具体情况
- 提供专业建议
- 推荐相关产品
- 建议就医情况（如需）

##### 产品咨询类

**可以咨询的问题：**
- "新手适合使用什么样的产品？"
- "XX产品和XX产品有什么区别？"
- "如何清洁保养我的设备？"
- "这个产品有什么使用注意事项？"

**AI会：**
- 分析您的需求和偏好
- 推荐合适的产品
- 详细介绍产品特点
- 提供使用指导

##### 情感陪伴类

**可以交流的话题：**
- 分享日常生活中的烦恼
- 讨论人际关系问题
- 表达内心感受
- 寻求情感支持

**AI会：**
- 耐心倾听您的倾诉
- 给予情感支持
- 提供建议和安慰
- 保护您的隐私

##### 知识科普类

**可以咨询的问题：**
- "什么是凯格尔运动？"
- "如何保持私密部位健康？"
- "健康饮食的基本原则是什么？"
- "如何建立健康的生活习惯？"

**AI会：**
- 解释专业术语
- 提供科学知识
- 推荐学习资源
- 纠正错误认知

#### 3.4 对话技巧

##### 描述清晰

**好的提问示例：**
- "我是一名程序员，每天工作10小时，最近感觉腰部酸痛，有什么缓解建议吗？"
- "我是第一次使用这类产品，想从基础的开始，有什么推荐吗？"

**不够好的提问示例：**
- "我身体不舒服"
- "推荐个东西"

##### 提供背景信息

为了让AI给出更准确的建议，请提供相关背景：

- 您的基本情况（年龄、职业）
- 具体症状或需求
- 已尝试过的解决方法
- 特殊需求或顾虑

##### 循序渐进

复杂问题可以分多次提问：

1. 先问："什么是X？"
2. 再问："X适合什么样的人？"
3. 最后问："我适合X吗？"

#### 3.5 智能推荐功能

AI伴侣会根据您的对话内容，智能推荐相关内容：

**推荐类型：**
- 相关产品
- 知识文章
- 使用教程
- 健康建议

**推荐原则：**
- 基于您的实际需求
- 不强制推销
- 尊重您的选择
- 保护隐私

#### 3.6 隐私保护

##### 对话隐私

- AI对话内容完全加密存储
- 仅您本人可以查看历史对话
- 支持清空对话记录
- 不向第三方共享对话内容

##### 清空对话记录

1. 点击右上角「清空」图标
2. 确认清空操作
3. 所有对话记录将被永久删除

**注意：** 清空后无法恢复，请谨慎操作。

##### 隐私模式下的AI

在隐身模式或伪装模式下：
- AI对话自动使用模糊化语言
- 产品推荐使用虚拟名称
- 不记录具体细节

#### 3.7 常见问题

**Q: AI伴侣的回答可靠吗？**

A: AI伴侣基于海量的专业医学和健康知识库训练，但AI建议仅供参考，不能替代专业医疗诊断。如有严重健康问题，请及时就医。

**Q: AI会记住我说过的话吗？**

A: AI会记住当前对话的上下文，以便更好地回答您的问题。但您随时可以清空对话记录。

**Q: 对话有次数限制吗？**

A: 普通用户每天可进行50次对话，会员用户无限制。

**Q: AI伴侣支持多语言吗？**

A: 目前支持中文和英文，未来会支持更多语言。
