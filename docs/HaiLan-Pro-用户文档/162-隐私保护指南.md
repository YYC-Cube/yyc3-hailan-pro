### 3. 隐私保护指南

#### 3.1 隐私保护概述

海蓝深知隐私对您的重要性。我们采用业界领先的隐私保护技术，确保您的个人信息和健康数据得到最严格的保护。

**核心保护措施：**

| 保护措施 | 说明 |
|----------|------|
| 双重加密 | 传输SSL加密 + 存储AES-256加密 |
| 数据脱敏 | 敏感信息自动脱敏显示 |
| 匿名认证 | 支持匿名账户使用 |
| 隐私模式 | 三级隐私模式切换 |
| 伪装发货 | 快递单信息虚拟化 |
| 浏览保护 | 无痕浏览模式 |

#### 3.2 隐私等级详解

##### 标准模式 (STANDARD)

**适用场景：** 私密环境使用，如个人手机、家庭电脑

**数据展示规则：**
- 邮箱：完整显示 `example@hailan.pro`
- 手机号：完整显示 `13800138000`
- 订单商品：真实商品名称
- 浏览历史：完整记录
- 收货地址：真实地址

**数据存储：**
- 所有数据正常存储
- 支持数据导出
- 支持账户删除

##### 隐身模式 (STEALTH)

**适用场景：** 公共场合或共享设备

**数据展示规则：**
- 邮箱：脱敏显示 `***@hailan.pro`
- 手机号：脱敏显示 `138****8000`
- 订单商品：虚拟化商品名称
- 浏览历史：不记录
- 收货地址：脱敏显示

**特殊功能：**
- 自动清除浏览痕迹
- 截屏自动模糊处理
- 切换应用时自动锁屏
- 通知内容虚拟化

**启用方法：**
```
我的 → 设置 → 隐私设置 → 选择「隐身模式」
```

##### 伪装模式 (DISGUISE)

**适用场景：** 最高隐私需求

**数据展示规则：**
- 用户昵称：虚拟化显示「用户88***」
- 所有列表项：完全虚拟化内容
- 订单详情：虚拟化所有敏感信息
- 收货地址：使用虚拟仓地址

**特殊功能：**
- 应用图标伪装为「健康管理」
- 通知伪装为「系统更新」
- 所有数据完全虚拟化
- 支持紧急退出（双击电源键）

**启用方法：**
```
我的 → 设置 → 隐私设置 → 选择「伪装模式」
→ 输入二次确认密码 → 确认启用
```

#### 3.3 数据安全保护

##### 传输安全

- **HTTPS加密**：所有数据传输采用HTTPS/TLS 1.3加密
- **证书固定**：防止中间人攻击
- **请求签名**：API请求签名验证

##### 存储安全

- **数据库加密**：敏感字段AES-256加密存储
- **密钥管理**：密钥定期轮换，独立存储
- **访问控制**：严格的数据库访问权限控制
- **备份加密**：数据备份文件加密存储

##### 账户安全

- **密码加密**：采用bcrypt哈希算法，不可逆加密
- **登录保护**：
  - 支持指纹/面部识别登录
  - 异常登录检测
  - 多次失败自动锁定
- **会话管理**：
  - Token自动过期
  - 支持强制下线所有设备
  - 单设备登录限制

#### 3.4 隐私设置操作指南

##### 修改隐私等级

1. 进入「我的」→「设置」
2. 点击「隐私设置」
3. 选择目标隐私等级
4. 阅读模式说明
5. 确认切换

**注意事项：**
- 切换隐私等级不会删除数据
- 伪装模式需要输入密码确认
- 切换后立即生效

##### 开启浏览保护

1. 进入「我的」→「设置」
2. 点击「浏览保护」
3. 开启「无痕浏览」
4. 选择保护级别：
   - 基础保护：不记录浏览历史
   - 高级保护：不记录+自动清理缓存
   - 极致保护：不记录+清理+截图保护

##### 设置通知隐私

1. 进入「我的」→「设置」
2. 点击「通知设置」
3. 选择通知内容显示级别：
   - 完整显示：显示所有通知内容
   - 简化显示：仅显示通知类型
   - 隐藏内容：仅显示「您有新消息」

##### 退出登录清理

1. 进入「我的」→「设置」
2. 点击「安全设置」
3. 开启「退出时清理数据」
4. 选择清理内容：
   - 浏览缓存
   - 图片缓存
   - 搜索历史
   - 表单数据

#### 3.5 数据管理权利

您对自己的数据拥有完全的控制权：

##### 查看个人数据

1. 进入「我的」→「设置」
2. 点击「数据管理」
3. 点击「查看我的数据」
4. 系统将生成数据报告并发送到您的邮箱

##### 导出个人数据

1. 进入「我的」→「设置」
2. 点击「数据管理」
3. 点击「导出数据」
4. 选择导出范围：
   - 基本信息
   - 订单记录
   - 健康数据
   - AI对话记录
5. 点击「确认导出」
6. 数据将通过加密邮件发送

##### 删除个人数据

**账户注销：**
1. 进入「我的」→「设置」
2. 点击「账户安全」
3. 滚动到底部，点击「注销账户」
4. 阅读注销须知
5. 确认注销

**注销前请注意：**
- 注销后数据无法恢复
- 未完成订单将取消
- 会员权益无法保留
- 建议先导出重要数据
