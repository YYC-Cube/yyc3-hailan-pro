### 1. 背景与目标

HaiLan Pro (海蓝) 是新一代高端、私密、智能的情趣健康生活管理平台。项目基于「五高五标五化」理念，通过 PWA 技术结合 AI 智能辅助与物联网，为用户提供从生理健康到心理愉悦的全方位解决方案。

#### 1.2 项目愿景
打造极致隐私、智能陪伴、品质合规、全场景覆盖的情趣健康生活管理平台，为用户提供安全、专业、高端的健康生活体验。

#### 1.3 核心价值主张
- **极致隐私**：双重加密、隐私浏览模式及伪装发货机制

- **品质合规**：医疗级标准商品，高端"海蓝蓝"视觉调性

### 3. 合规性审核清单

#### 3.1 个人信息保护法合规

HaiLan Pro 处理用户敏感个人信息，需严格遵守《个人信息保护法》相关要求。

```yaml
个人信息保护合规清单:

  数据收集合规:
    - [ ] 明确告知用户数据收集目的、方式和范围
    - [ ] 获得用户明确同意（非默认勾选）
    - [ ] 提供隐私政策完整文本
    - [ ] 隐私政策内容清晰易懂
    - [ ] 最小化收集原则（只收集必要信息）
    - [ ] 区分必要信息和可选信息

  敏感个人信息处理:
    - [ ] 健康数据属于敏感个人信息，需单独同意
    - [ ] 生理特征数据（如使用习惯）需特殊保护
    - [ ] 性生活相关数据属于高度敏感信息
    - [ ] 敏感数据加密存储
    - [ ] 敏感数据访问权限严格控制
    - [ ] 敏感数据留存期限明确（不超过必要期限）

  用户权利保障:
    - [ ] 支持用户查询个人信息
    - [ ] 支持用户更正个人信息
    - [ ] 支持用户删除个人信息（注销账户）
    - [ ] 支持用户撤回同意
    - [ ] 支持用户获取个人信息副本
    - [ ] 提供便捷的权利行使渠道

  数据跨境传输:
    - [ ] 评估数据跨境必要性
    - [ ] 获得用户单独同意
    - [ ] 确保接收方具备保护能力
    - [ ] 签订标准合同条款
    - [ ] 向监管机构备案

  第三方共享:
    - [ ] 向第三方共享需获得同意
    - [ ] 明确告知接收方信息
    - [ ] 监督接收方保护义务履行
    - [ ] 提供拒绝共享的选项
```

#### 3.9 合规审核流程

```typescript
// __tests__/compliance/compliance-check.test.ts
import { describe, it, expect } from 'bun:test';
import { ComplianceChecker } from '@/services/compliance.service';

describe('合规性检查测试', () => {
  let checker: ComplianceChecker;

  beforeEach(() => {
    checker = new ComplianceChecker();
  });

  describe('隐私政策检查', () => {
    it('隐私政策应包含所有必要条款', async () => {
      const policyContent = await fetchPrivacyPolicy();
      const requiredSections = [
        '收集信息类型',
        '使用目的',
        '共享规则',
        '用户权利',
        'Cookie政策',
        '联系方式',
      ];

      const result = checker.checkPrivacyPolicy(policyContent);
      expect(result.hasAllSections).toBe(true);
      expect(result.missingSections).toHaveLength(0);
    });

    it('隐私政策应在注册时显示', async () => {
      const result = await checker.checkRegistrationFlow();
      expect(result.privacyPolicyShown).toBe(true);
      expect(result.userMustAgree).toBe(true);
      expect(result.notAgreeByDefault).toBe(true);
    });
  });

  describe('用户同意检查', () => {
    it('敏感数据收集需单独同意', async () => {
      const consentFlow = await checker.checkHealthDataConsent();

      expect(consentFlow.separateConsent).toBe(true);
      expect(consentFlow.informative).toBe(true);
      expect(consentFlow.revocable).toBe(true);
    });

    it('用户应能撤回同意', async () => {
      const result = await checker.checkConsentRevocation();
      expect(result.canRevoke).toBe(true);
      expect(result.dataDeleted).toBe(true);
    });
  });

  describe('数据安全检查', () => {
    it('敏感数据应加密存储', async () => {
      const result = await checker.checkDataEncryption();
      expect(result.passwordHashed).toBe(true);
      expect(result.sensitiveDataEncrypted).toBe(true);
      expect(result.transmissionEncrypted).toBe(true);
    });

    it('应有数据访问控制', async () => {
      const result = await checker.checkAccessControl();
      expect(result.hasAuthentication).toBe(true);
      expect(result.hasAuthorization).toBe(true);
      expect(result.logsAccess).toBe(true);
    });
  });

  describe('用户权利检查', () => {
    it('用户应能导出个人数据', async () => {
      const result = await checker.checkDataExport();
      expect(result.canExport).toBe(true);
      expect(result.formatReadable).toBe(true);
    });

    it('用户应能删除账户', async () => {
      const result = await checker.checkAccountDeletion();
      expect(result.canDelete).toBe(true);
      expect(result.consequencesInformed).toBe(true);
      expect(result.dataWiped).toBe(true);
    });
  });
});
```
