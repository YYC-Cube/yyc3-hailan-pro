### 3. 用户验收测试规范

#### 3.1 UAT测试概述

用户验收测试(User Acceptance Test, UAT)是发布前的最终测试阶段，由真实用户或代表用户的测试人员执行，验证系统是否满足业务需求。

```yaml
UAT测试目标:

  业务验证:
    - 验证核心业务流程可用
    - 验证业务规则正确执行
    - 验证用户需求被满足
    - 验证系统支持实际工作场景

  用户体验验证:
    - 验证界面友好易用
    - 验证操作流程顺畅
    - 验证响应时间可接受
    - 验证错误提示清晰

  场景验证:
    - 验证真实使用场景
    - 验证边界条件处理
    - 验证异常情况处理
    - 验证数据准确性
```

#### 3.5 UAT反馈收集

```typescript
// uat/feedback-collector.ts
interface UATFeedback {
  scenarioId: string;
  userId: string;
  rating: number; // 1-5星
  issues: string[];
  suggestions: string[];
  canRelease: boolean;
  comments: string;
}

class UATFeedbackCollector {
  async collectFeedback(feedback: UATFeedback): Promise<void> {
    // 存储反馈
    await this.saveFeedback(feedback);

    // 如果评分低，自动通知相关人员
    if (feedback.rating <= 2) {
      await this.notifyTeam(feedback);
    }
  }

  async generateReport(): Promise<UATReport> {
    const feedbacks = await this.getAllFeedbacks();

    return {
      totalScenarios: feedbacks.length,
      executedScenarios: feedbacks.filter(f => f.rating > 0).length,
      averageRating: this.calculateAverageRating(feedbacks),
      issues: this.collectAllIssues(feedbacks),
      suggestions: this.collectAllSuggestions(feedbacks),
      canRelease: this.assessReleaseReadiness(feedbacks),
    };
  }

  private calculateAverageRating(feedbacks: UATFeedback[]): number {
    const valid = feedbacks.filter(f => f.rating > 0);
    if (valid.length === 0) return 0;
    const sum = valid.reduce((acc, f) => acc + f.rating, 0);
    return sum / valid.length;
  }

  private assessReleaseReadiness(feedbacks: UATFeedback[]): boolean {
    const avgRating = this.calculateAverageRating(feedbacks);
    const cannotReleaseCount = feedbacks.filter(f => !f.canRelease).length;

    return avgRating >= 4 && cannotReleaseCount === 0;
  }
}
```
